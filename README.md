Video Communication Analyzer — PhaseZero Assessment (Nov 2025)



This project processes a public video (YouTube or MP4), extracts audio, transcribes speech using faster-whisper, and generates:



Clarity Score (0–100%)



Communication Focus (one-line topic summary)



This version is 100% free, requires no OpenAI, uses offline transcription, and optionally connects to DeepSeek API for enhanced scoring.
